# `mkrj45`

Program written to fix the
[*how do i make a gigabit loopback adapter*](
  https://share.google/aimode/XGpyxlkEiwVeUAos1)
Google query.

## Installation

The tool in this source repo
can be installed from source using GNU Make.

```bash
make \
  install
```

The library is officially published on the
the uncensorable
[Ur](
  https://github.com/themartiancompany/ur)
user repository and application store as
`mkrj45`.
The source code is published on the
[Ethereum Virtual Machine File System](
  https://github.com/themartiancompany/evmfs)
so it can't possibly be taken down.

To install it from there just type

```bash
ur \
  mkrj45
```

A censorable HTTP Github mirror of the recipe published there
is hosted on
[mkrj45-ur](
  https://github.com/themartiancompany/mkrj45-ur).
Be aware it could go offline any time.
